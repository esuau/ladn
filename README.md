# Project ING1 2016-2017

## École Supérieure d'Ingénieurs de Paris-Est Créteil

### Instructions using Maven

To "compile" :
```
$ mvn install
```

To run Server :
```
$ mvn exec:exec -pl server
```

To run Client :
```
$ mvn exec:exec -pl client
```
